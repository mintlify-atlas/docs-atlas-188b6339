## How It Works

<Steps>
  <Step title="Background Monitoring">
    The extension monitors active tabs via the Browser Tabs API, detecting window focus and idle state changes
  </Step>
  <Step title="Local Storage">
    All tracking data is stored in IndexedDB using Dexie.js with three layers: raw history logs, site aggregates, and page details
  </Step>
  <Step title="Analytics Dashboard">
    View your activity in the popup with total time, daily/hourly trends, and top sites ranked by duration
  </Step>
</Steps>

## Tech Stack

Built with modern web technologies for performance and reliability:

- **WXT** — Modern web extension framework with HMR support
- **Vue 3** — Progressive JavaScript framework with Composition API
- **Dexie.js** — Wrapper library for IndexedDB with reactive queries
- **Tailwind CSS 4** — Utility-first CSS framework
- **Arktype** — Runtime type validation
- **Vitest** — Unit test framework
